一个英文翻译小程序

这是一个用Python语言编写的小程序，支持中英文翻译。目前，只能使用这些功能。

目的是创建一个可以在终端上进行翻译的程序，因为我经常在终端上工作，不想去网上搜索，我可以比较中英文来帮助学习中英文。

以下是如何在终端中使用它的方法：

在文件开头添加：
```python
#!/usr/bin/python3
```

在终端执行：
```shell
sudo chmod +x min.py
```
授予可执行权限，然后执行下一步：
```shell
sudo mv min.py /usr/local/bin
```
将它重命名：
```shell
sudo mv min.py min
```
试一试吧！
